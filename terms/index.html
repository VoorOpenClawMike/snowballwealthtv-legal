Terms of Service

Last updated: April 7, 2026

1. Service Description

SnowballWealthTV is an automated system that repurposes YouTube finance video content and publishes it to TikTok using the TikTok Content Posting API.

2. Authorization

By connecting your TikTok account, you authorize SnowballWealthTV to post videos on your behalf via OAuth 2.0. You can revoke this access at any time via TikTok Settings → Security → Authorized Apps.

3. Content

All published content originates from your own YouTube channel. SnowballWealthTV does not create original content — it automates the repurposing workflow.

4. Data Use

We only store the OAuth access token required to post videos. No personal data is sold or shared with third parties.
